# Bible Quest - Interactive Bible Trivia Game

An engaging, interactive web-based trivia game that tests and expands players' knowledge of the Bible.

## Features

- **Multiple Difficulty Levels**: Choose from Easy, Medium, or Hard questions
- **56 Curated Questions**: Carefully crafted questions covering Old Testament, New Testament, and Gospels
- **Beautiful Design**: Warm biblical theme with navy, gold, and cream colors
- **Scripture References**: Each answer includes the relevant Bible verse reference
- **Responsive Design**: Works perfectly on mobile, tablet, and desktop
- **Smooth Animations**: Engaging transitions and feedback
- **Score Tracking**: Real-time score tracking and detailed results breakdown
- **No Dependencies**: Runs entirely in the browser (except React CDN)

## How to Play

1. Select your difficulty level (Easy, Medium, or Hard)
2. Click "Start Quest" to begin
3. Answer 10 randomly selected questions
4. Get immediate feedback with Scripture references
5. View your final score and statistics

## Running Locally

Simply open `index.html` in your web browser. No build process or installation required!

Or use a local server:

```bash
# Python 3
python -m http.server 8000

# Then visit: http://localhost:8000/index.html
```
